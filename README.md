# Adarsh Naidu - Executive Portfolio Website

A premium, executive-style portfolio website built with Next.js, TypeScript, and Tailwind CSS, showcasing the professional achievements and expertise of Adarsh Naidu, Chief Data Architect and AWS Principal Solution Architect.

## Features

- **Executive Design**: Sophisticated dark theme with gold accents
- **Responsive Layout**: Fully responsive across all devices
- **Modern Tech Stack**: Built with Next.js 14, TypeScript, and Tailwind CSS
- **Smooth Animations**: Professional animations and transitions
- **SEO Optimized**: Proper meta tags and semantic HTML

## Sections

1. **Hero Section**: Professional profile photo with dynamic introduction and key metrics
2. **About**: Executive profile and career evolution
3. **Achievements**: Career highlights with measurable impact
4. **Experience**: Detailed professional journey with timeline
5. **Skills**: Comprehensive technical expertise showcase
6. **Blog & Press**: Published articles and media coverage
7. **Testimonials**: 15+ LinkedIn recommendations with interactive carousel
8. **Education**: Academic credentials and certifications
9. **Contact**: Multiple contact methods and CTA

## Adding Your Profile Photo

The Hero section features a professional profile photo on the left side. To add your photo:

1. Save your professional headshot as `profile.jpg`
2. Place it in the `public` folder
3. Recommended: 800x800 pixels or larger, under 500KB

For detailed instructions, see [HOW_TO_ADD_PHOTO.md](./HOW_TO_ADD_PHOTO.md)

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## Deployment

This website can be deployed to:

- **Vercel** (Recommended): `vercel deploy`
- **Netlify**: Connect to Git repository
- **AWS Amplify**: Deploy via Git or CLI
- **Any static hosting**: Export with `npm run build`

## Technology Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Fonts**: Inter, Playfair Display (Google Fonts)

## Customization

### Colors

Edit `tailwind.config.ts` to customize the color scheme:
- `executive-dark`: #0a1628
- `executive-navy`: #1e3a5f
- `executive-gold`: #d4af37

### Content

Update content in the following component files:
- `components/Hero.tsx`
- `components/About.tsx`
- `components/Achievements.tsx`
- `components/Experience.tsx`
- `components/Skills.tsx`
- `components/Education.tsx`
- `components/Contact.tsx`
